# Student Foundation

A web application for managing and browsing academic theses.

## Features

- **Home Page** - Welcome section with latest theses
- **Archive** - Browse all available theses
- **Thesis Details** - View detailed information about each thesis
- **Admin Dashboard** - Manage theses content

## Tech Stack

- React 18
- React Router
- Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
```

## Deployment
